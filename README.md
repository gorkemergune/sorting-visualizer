Step-by-step sorting algorithm visualizer. Watch classic algorithms sort arrays in real time — pause, step forward/backward, adjust speed and array size.

---

## Algorithms

| Algorithm | Best | Average | Worst | Stable |
| :--- | :--- | :--- | :--- | :--- |
| Bubble Sort | $O(n)$ | $O(n^2)$ | $O(n^2)$ | yes |
| Selection Sort | $O(n^2)$ | $O(n^2)$ | $O(n^2)$ | no |
| Insertion Sort | $O(n)$ | $O(n^2)$ | $O(n^2)$ | yes |
| Merge Sort | $O(n \log n)$ | $O(n \log n)$ | $O(n \log n)$ | yes |
| Quick Sort | $O(n \log n)$ | $O(n \log n)$ | $O(n^2)$ | no |
| Heap Sort | $O(n \log n)$ | $O(n \log n)$ | $O(n \log n)$ | no |
